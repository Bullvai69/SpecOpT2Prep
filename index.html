Program
Adapted NSW Physical Training Guide
Structure — 8-week block + break
Mon Upper A (push) · Tue HIIT kettlebell/calisthenics · Wed Legs · Thu Upper B (pull) · Fri HIIT endurance + movement · Sat long endurance + movement · Sun full rest.

3 gym sessions (push / pull / legs), two HIIT sessions that alternate in style — Tuesday is kettlebell/calisthenics conditioning, Friday is endurance intervals — Big 3 calisthenics on the lifting days, and ground-based movement work on the endurance days. Progressions follow NSW Tables 1, 2 and 6.
Two HIIT styles
Tue — Kettlebell / calisthenics circuit: power-endurance conditioning. Kettlebells are gym-only, so each exercise shows a bodyweight "or" version for home. Fri — Endurance intervals: pick run, swim or road bike, pace-matched per NSW Table 1 with swim distances in meters rounded to pool lengths (25m).
Movement / mobility
Ground-based movement work (joint CARs, deep squat sits, hanging, crawling, hip flows) in the LeoMoves tradition, placed on the endurance days. Each strength day also opens with a 2-4 min mobility prep for the joints you're about to load. These keep joints resilient — quality and control over range.
8-week mesocycle + break
Foundation (wk 1-2) → Build (wk 3-4) → Intensify (wk 5-6) → Peak (wk 7) → Back-off (wk 8) → Break (wk 9, marked with a dot; load eases ~30% or light movement only). At the end of the cycle, we rebuild the next 8 weeks from the weights and reps you logged.
Updating this app
Replace index.html in your GitHub repo with a new version any time the program changes. Your logged data stays on this device — it's stored locally and isn't affected by the file swap.
